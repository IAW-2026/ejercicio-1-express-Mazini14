¿Qué es Este Proyecto?

Este es un ejercicio educativo diseñado para aprender a trabajar con Express.js y el manejo de múltiples rutas en Node.js.

🎯 Objetivo del Ejercicio

El propósito es comprender cómo un servidor Express puede manejar diferentes rutas (/, /acerca, /contacto) y servir contenido HTML distinto para cada una. También demuestra la importancia de conectar las páginas entre sí con navegación clara.

📚 Tecnologías Utilizadas

  • Node.js - Entorno de ejecución JavaScript
  • Express.js - Framework web para Node.js
  • HTML5 - Estructura y semántica web
  • CSS3 - Diseño y estilos responsivos
  • JavaScript - Interactividad del lado del cliente

🔧 Funcionalidades Clave

  • ✅ Manejo de múltiples rutas GET
  • ✅ Servicio de archivos estáticos con express.static()
  • ✅ Navegación fluida entre páginas
  • ✅ Manejo de errores 404
  • ✅ Diseño responsivo y moderno

💡 Lo Que Aprenderas

Con este ejercicio aprenderás a:

  1. Crear un servidor Express funcional
  2. Definir múltiples rutas en una aplicación
  3. Servir archivos estáticos (CSS, JavaScript)
  4. Conectar páginas HTML con navegación
  5. Manejar errores y rutas no encontradas (404)
  6. Organizar un proyecto Express de manera profesional
