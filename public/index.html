Este sitio demuestra cómo Express.js puede servir múltiples rutas y conectar diferentes páginas HTML entre sí.

✨ Características del Sitio

📝 Rutas Múltiples

Navega entre diferentes páginas utilizando rutas Express bien definidas.

🎨 Diseño Responsivo

El sitio se visualiza correctamente en cualquier dispositivo.

🔗 Navegación Integrada

Accede fácilmente a todas las secciones desde cualquier página.

📦 Archivos Estáticos

Estilos CSS y JavaScript servidos desde la carpeta public.

📖 Cómo Funciona

Este servidor utiliza express.static() para servir archivos estáticos (CSS, JavaScript) y rutas GET para servir cada página HTML:

  • GET / → Sirve index.html
  • GET /acerca → Sirve acerca.html
  • GET /contacto → Sirve contacto.html
