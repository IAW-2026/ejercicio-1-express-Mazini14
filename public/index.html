Bienvenido al Servidor Express

Este es un ejemplo de servicio de archivos estáticos con express.static()

¿Qué es express.static?

es una función middleware de Express que sirve archivos estáticos (HTML, CSS, JavaScript, imágenes, etc.) desde un directorio especificado.

Archivos Servidos

  • 📄 index.html - Este archivo HTML
  • 🎨 styles.css - Estilos CSS
  • ⚙️ script.js - JavaScript interactivo
